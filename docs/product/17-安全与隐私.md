# 安全与隐私

安全与隐私模块汇总 AIPP 在数据安全、隐私保护和内容防护方面的设计，确保用户数据安全可控。

---

## 本地存储优先

### 数据不离开设备
- 所有数据存储在本地 SQLite 数据库
- 无任何云端同步功能
- 用户完全控制数据的生命周期
- 数据格式开放，无数据锁定

### 无遥测功能
- 不收集用户使用数据
- 不发送任何分析或诊断数据
- 完全透明的开源代码
- 隐私优先的设计理念

---

## 防泄漏模式

### 敏感内容遮罩
- `AntiLeakageContext.tsx` 提供全局防泄漏上下文
- 开启后自动遮罩对话内容、消息和标题
- 防止屏幕分享、演示或旁人窥视时泄露敏感信息

### 临时显示控制
- `toggleReveal()` 支持临时显示被遮罩的内容
- `resetReveal()` 一键重置为遮罩状态
- 关闭防泄漏模式时自动重置所有显示状态

### 全局配置
- 通过功能配置（feature_config）全局开关
- 设置界面中可启用/禁用
- 影响 ConversationTitle、MessageItem、ConversationUI 等多个核心组件

---

## 操作权限控制

### 权限请求机制
- AI 工具调用（文件操作、Shell 执行）前需用户授权
- `OperationPermissionDialog` 权限请求对话框
- 显示操作详情和风险提示
- 用户可选择允许或拒绝

### 权限规则配置
- 按操作类型配置权限（读取、写入、执行等）
- 按路径/命令白名单自动审批
- 全局默认权限设置
- 权限规则可编辑和重置

### 权限持久化
- 权限决策记录存储
- 历史权限审计
- 权限统计分析

---

## 加密分享

### Provider 加密分享
- LLM 提供商配置使用 AES-256-GCM 加密分享
- 分享码通过 gzip 压缩 + Base64 编码
- 导入时需要输入密码解密
- 保护 API Key 等敏感凭据不泄露

### 助手安全分享
- 助手配置使用压缩编码分享（不含敏感信息）
- 分享码不包含 API Key 或 Provider 凭据
- 安全的配置迁移和协作

---

## 用户数据掌控

### 数据备份与恢复
- 用户可直接备份 SQLite 数据库文件
- 数据库文件可跨设备迁移
- 标准 SQLite 格式，可用第三方工具查看

### 数据删除
- 用户可随时删除对话、助手、模型等数据
- 级联删除确保关联数据清理
- 完全删除不留残留
