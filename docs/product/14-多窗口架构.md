# 多窗口架构

AIPP 采用多窗口架构设计，不同功能使用独立的 Tauri 窗口，提供更好的用户体验和任务隔离。

---

## Ask 窗口

### 全局快捷键快速打开
- 标签：`ask`
- 全局快捷键 `Ctrl+Shift+I` 快速打开
- 简洁的提问界面
- 快速 AI 查询

### 简洁 AI 查询界面
- `AskWindow.tsx` Ask 窗口入口
- `AskWindowPrepare.tsx` 准备组件
- 精简的输入框
- 快速响应展示

### 快速提问与响应
- 支持快速提问
- 创建新对话或附加到现有对话
- 响应快速展示
- 窗口自动关闭选项

---

## ChatUI 窗口

### 主聊天界面
- 标签：`chat_ui`
- `ChatUIWindow.tsx` ChatUI 窗口入口
- 完整的对话功能
- 消息列表和输入

### 完整对话功能
- `ConversationUI.tsx` 对话 UI 组件
- 消息展示和交互
- 工具调用展示
- 制品预览嵌入

### 消息列表与输入
- `ConversationList.tsx` 对话列表
- `MessageItem.tsx` 消息项
- 输入框组件
- 文件拖拽上传

### MCP 服务器配置

- 工具列表展示

## Artifact Preview 窗口

### 内容预览专用窗口
- 标签：`artifact_preview`
- `ArtifactPreviewWindow.tsx` 制品预览窗口入口
- 独立的预览界面
- 可调整大小

### HTML/SVG/React/Vue 渲染
- `preview_router.rs` 预览路由
- HTML 内容渲染
- SVG 矢量图形渲染
- React 组件预览
- Vue 组件预览

### 独立窗口展示
- 全屏预览支持
- 多实例预览
- 预览状态管理
- 窗口关闭确认

## 窗口管理

### App.tsx 窗口路由
- `App.tsx` 主应用组件
- 懒加载窗口组件
- 根据窗口标签动态加载
- 窗口生命周期管理

### 窗口操作
- `window.rs` 窗口管理
- 窗口创建和关闭
- 窗口显示和隐藏
- 窗口大小和位置管理

---

## 移动端适配

### 单窗口模式
- 移动端使用单窗口模式
- 所有功能在一个窗口中
- 页面切换导航
- 响应式布局

---
相关源码:
- `src/windows/` - 各窗口入口组件
- `src/App.tsx` - 窗口路由器
- `src-tauri/src/window.rs` - 窗口管理
- `src-tauri/src/lib.rs` - 窗口注册
