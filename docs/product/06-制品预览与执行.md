# 制品预览与执行

制品预览与执行模块允许 AI 生成的内容（HTML、SVG、XML、Markdown、Mermaid、Drawio、React/Vue 组件、Python 脚本等）在独立窗口中实时预览和执行。

---

## HTML/SVG/XML 预览

### HTML 内容预览
- 自动检测消息中的 HTML 代码块
- 支持内联样式和外部资源引用
- 在 Artifact Preview 窗口中渲染
- 实时更新内容变化

### SVG 渲染
- 支持 SVG 代码块渲染
- 矢量图形无损显示
- 支持动画 SVG

### XML 渲染
- 支持 XML 代码块格式化显示
- XML 语法高亮
- 结构化展示

### Markdown/Mermaid 渲染
- 支持 Markdown 文档渲染
- 支持 Mermaid 图表渲染
- 流程图、时序图、甘特图等
- 实时渲染更新

### Drawio 图表渲染
- 支持 Drawio 图表渲染
- Drawio 图表文件预览
- 矢量图形显示

### 独立预览窗口
- Artifact Preview 专用窗口
- 可调整大小
- 支持全屏预览

---

## React 组件预览与执行

### React 组件代码解析
- 自动识别 React 组件代码
- 解析组件定义和依赖
- 提取导出的组件

### 实时预览
- `react_preview.rs` 处理 React 预览
- 组件代码动态编译
- 实时渲染展示

### 组件运行与交互
- `react_runner.rs` 运行 React 组件
- 支持组件交互（点击、输入等）
- 状态保持

### 共享组件模板
- `shared_components.rs` 提供共享组件
- 常用 UI 组件预置
- 模板复用

---

## Vue 组件预览与执行

### Vue 组件代码解析
- 自动识别 Vue 单文件组件（SFC）
- 解析 template、script、style
- 支持 Composition API 和 Options API

### 实时预览
- `vue_preview.rs` 处理 Vue 预览
- 组件代码动态编译
- 实时渲染展示

### 组件运行与交互
- `vue_runner.rs` 运行 Vue 组件
- 支持组件交互
- 响应式数据更新

---

## Python 脚本执行

### Python 脚本运行
- 检测 Python 代码块
- 使用系统 Python 或 UV 环境
- 执行脚本并捕获输出

### 环境检测与配置
- 检测 Python 安装
- UV 工具支持
- 虚拟环境管理

### 输出结果展示
- 标准输出捕获
- 错误输出展示
- 执行状态显示

## Bash/PowerShell/AppleScript 执行

## 制品桥接 API

### 数据库操作 API
- `artifact_db_query` - 制品数据库查询
- `artifact_db_execute` - 制品数据库执行
- `artifact_db_batch_execute` - 制品数据库批量执行
- `artifact_db_get_tables` - 制品数据库获取表
- `artifact_db_get_columns` - 制品数据库获取列
- `artifact_db_exists` - 制品数据库存在检查
- `artifact_db_delete` - 制品数据库删除
- `artifact_db_list` - 制品数据库列表

### AI 与助手 API
- `artifact_ai_ask` - 制品 AI 查询
- `artifact_get_assistants` - 制品获取助手
- `artifact_get_config` - 制品获取配置

### 预览路由
- `preview_router.rs` 路由预览请求
- 根据内容类型分发
- 统一预览接口

## 其他功能

### 代码工具
- `code_utils.rs` 代码处理工具
- 代码块提取
- 语言检测

### 制品统计与元数据
- `get_artifacts_statistics` - 获取制品统计
- `get_artifacts_for_completion` - 获取用于补全的制品
- `generate_artifact_metadata` - 生成制品元数据

### 制品编辑
- 支持编辑已保存的制品
- 版本历史

### 预览窗口管理
- `open_artifact_window` - 打开制品窗口
- `open_artifact_collections_window` - 打开制品集合窗口
- `restore_artifact_preview` - 恢复制品预览
- `save_artifact_to_collection` - 保存制品到集合

### 缓存管理
- `clear_react_artifact_cache` - 清除 React 制品缓存
- `clear_vue_artifact_cache` - 清除 Vue 制品缓存
- `clear_all_template_cache` - 清除所有模板缓存
