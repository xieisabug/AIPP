# 助手系统

助手系统允许用户创建和管理不同类型的 AI 助手，每个助手可以有独立的系统提示词、模型配置和工具设置。

---

## 助手类型

### 内置助手类型
- **0: 普通对话助手** - 标准的 AI 对话助手
- **4: ACP 助手** - 集成 Agent Client Protocol 的助手
- 数据库中预留的助手类型值：1=多模型对比, 2=工作流, 3=展示（通过插件扩展）

### 自定义助手类型（通过插件）
- 插件系统可扩展新的助手类型
- `AssistantType` 插件类型
- 动态注册助手类型
- 灵活的表单定制

### 助手类型表单渲染
- `AssistantFormRenderer.tsx` 助手表单渲染器
- 根据助手类型动态渲染不同表单
- React Hook Form 表单管理
- Zod 数据验证

---

## 助手配置表单

### 动态表单字段
- 表单字段按位置分组：header 和 body
- 每个字段可配置位置、标签、提示等
- 支持多种字段类型：string、number、select 等
- 插件可动态添加字段

### 表单验证
- React Hook Form + Zod 验证
- 必填字段验证
- 数据格式验证
- 友好的错误提示

### 字段位置配置
- **header 区域** - 助手基础信息（名称、描述等）
- **body 区域** - 详细配置（系统提示词、模型参数等）
- 位置可配置，灵活布局

---

## 系统提示词

### 助手系统提示词配置
- 每个助手可配置独立的系统提示词
- 系统提示词持久化存储
- 支持多行文本输入

### 提示词模板支持
- 可使用模板变量
- 运行时变量替换
- 灵活的提示词构建

### 运行时提示词构建
- 对话开始时合并系统提示词
- 支持用户输入附加提示词
- 最终提示词传递给 AI

---

## 助手运行

### 选择助手开始对话
- 对话创建时选择助手
- 对话与助手关联
- 助手切换影响新对话

### 助手配置与模型配置合并
- 助手模型配置优先级高于提供商默认配置
- 灵活的配置覆盖机制
- 最终配置传递给 genai 客户端

### 助手特定逻辑执行
- 不同助手类型有不同运行逻辑
- 插件可实现自定义运行逻辑
- `onAssistantTypeRun` 生命周期

---

## 助手 CRUD

### 助手管理 API
- `add_assistant` - 添加新助手
- `get_assistant` - 获取单个助手
- `get_assistants` - 获取助手列表
- `save_assistant` - 保存助手配置
- `delete_assistant` - 删除助手
- `copy_assistant` - 复制助手
- `export_assistant` - 导出助手
- `import_assistant` - 导入助手
- `get_assistant_field_value` - 获取助手字段值
- `get_acp_working_directory` - 获取 ACP 工作目录

### 助手模型配置
- `update_assistant_model_config_value` - 更新助手模型配置值
- 助手可配置多个模型
- 模型优先级排序
- 模型参数覆盖

### 助手 MCP 配置
- `update_assistant_mcp_config` - 更新助手 MCP 配置
- `update_assistant_mcp_tool_config` - 更新助手 MCP 工具配置
- `bulk_update_assistant_mcp_tools` - 批量更新助手 MCP 工具

### 助手 Skills 配置
- `get_assistant_skills` - 获取助手技能
- `get_enabled_assistant_skills` - 获取启用的助手技能
- `update_assistant_skill_config` - 更新助手技能配置
- `toggle_assistant_skill` - 切换助手技能
- `remove_assistant_skill` - 移除助手技能
- `bulk_update_assistant_skills` - 批量更新助手技能
- `cleanup_orphaned_skill_configs` - 清理孤立的技能配置

---

## 助手分享

### 压缩分享码
- 支持将助手配置导出为压缩分享码
- 使用 gzip 压缩 + Base64 编码
- 通过分享码可完整还原助手配置

### 分享与导入
- `export_assistant` 导出助手为分享码
- `import_assistant` 通过分享码导入助手
- `ShareDialog.tsx` 分享对话框 UI
- 一键复制分享码到剪贴板
