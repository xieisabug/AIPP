# 定时任务

定时任务模块允许用户创建和管理定时执行的任务，例如定时对话总结等。

---

## 定时任务管理

### 定时任务创建、编辑、删除
- `scheduled_task_api.rs` 定时任务 API
- `create_scheduled_task` - 创建定时任务
- `update_scheduled_task` - 更新定时任务
- `delete_scheduled_task` - 删除定时任务
- `list_scheduled_tasks` - 列出定时任务
- 任务名称、描述配置
- 任务启用/禁用开关

### 定时任务配置
- Cron 类调度（支持日/周/月）
- 任务类型选择
- 任务参数配置
- 执行时间设置

### 定时任务运行
- `run_scheduled_task_now` - 立即运行定时任务
- 手动触发任务执行

### 定时任务日志与运行记录
- `list_scheduled_task_logs` - 列出定时任务日志
- `list_scheduled_task_runs` - 列出定时任务运行记录
- 任务执行历史
- 执行状态追踪

### 定时任务列表展示
- Schedule 窗口展示任务列表
- 任务状态显示（启用/禁用）
- 下次执行时间显示
- 任务执行历史

---

## 对话总结任务

### 定时对话总结
- `summary_task.rs` 总结任务实现
- 定时触发对话总结
- 可配置总结的对话范围
- 总结结果保存

### 总结任务调度
- `scheduler/mod.rs` 调度器模块
- 任务队列管理
- 任务触发执行
- 任务失败重试

### 总结结果存储
- 总结结果持久化到数据库
- 对话与总结关联
- 总结历史查询
- 总结时间记录

---

## 调度器架构

### 调度器模块
- `scheduler/mod.rs` 调度器主模块
- `scheduled_task.rs` 定时任务定义
- `summary_task.rs` 总结任务实现
- 任务注册与管理

### 任务执行
- 异步任务执行
- 任务超时控制
- 任务错误处理
- 任务状态更新
