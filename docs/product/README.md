# AIPP 产品功能文档

> 本文档介绍 AIPP (AI 助手平台) 已实现的所有功能，用于项目讨论和协作参考。

## 产品简介

AIPP 是一个基于 Tauri 2.0 构建的跨平台桌面 AI 助手平台，支持多模型 AI 对话、MCP 工具调用、内容预览、脚本执行等功能。所有数据本地存储，保护用户隐私。

## 技术栈概览

- **后端**: Rust + Tauri 2.0 + SQLite
- **前端**: React 19 + TypeScript + Vite
- **UI**: shadcn/ui + Radix UI + Tailwind CSS
- **AI**: genai 客户端 + MCP 协议
- **其他**: Tokio 异步运行时
