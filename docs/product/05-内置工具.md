# 内置工具

内置工具是 MCP 工具系统的一部分，提供了一系列开箱即用的 MCP 工具，包括 Web 搜索、文件操作、Shell 执行、Agent 能力、UI 交互、Artifact 管理、动态 MCP 加载等常用功能。内置工具与外部 MCP 服务器统一管理，均使用 `aipp:` 前缀标识。

---

## Web 搜索（多引擎）

### 搜索引擎支持
- 支持多种搜索引擎：
  - **Google** - 谷歌搜索
  - **Bing** - 必应搜索
  - **DuckDuckGo** - 隐私优先搜索引擎
  - **Kagi** - 付费搜索引擎

### 智能回退策略
- 引擎管理器 `engine_manager.rs` 管理多个搜索引擎
- 主引擎失败时自动回退到备用引擎
- 默认策略：Google → Bing 回退
- 可配置引擎优先级

### 搜索结果获取与格式化
- `handler.rs` 主搜索处理器
- 搜索结果统一格式化
- 结果摘要生成
- 链接与标题提取

## URL 内容获取

### Chrome DevTools Protocol 浏览器自动化
- `chromiumoxide/` 集成 Chrome DevTools Protocol（CDP）
- `browser_pool.rs` 浏览器池管理
- `fetcher.rs` 内容获取器
- 支持无头浏览器模式
- 页面等待与元素选择

### HTTP 客户端回退
- 浏览器获取失败时使用 HTTP 客户端回退
- reqwest 库实现 HTTP 请求
- 多种内容类型支持
- 超时与重试机制

### 内容抓取与清理
- HTML 内容解析与清理
- 去除广告和无关元素
- 主要文本内容提取
- 内容截断与格式化

### 指纹管理（反检测）
- `fingerprint.rs` 指纹管理
- 浏览器指纹随机化
- User-Agent 轮换
- 反检测措施

## Shell 执行

### Bash 命令执行
- Linux/macOS Bash 脚本执行
- `bash_ops.rs` Bash 操作实现
- 命令参数解析
- 工作目录设置

### 命令结果返回
- 标准输出捕获
- 错误输出捕获
- 退出码返回
- 执行超时控制

### 权限请求与审批
- 操作前权限确认
- `OperationPermissionDialog` 权限请求对话框
- 权限决策持久化
- 自动审批配置
- 支持三种决策：`allow`（本次允许）、`allow_and_save`（允许并记住）、`deny`（拒绝）
- ACP 工具调用权限独立确认流程（`confirm_acp_permission`）

## UI交互工具

### 向用户提问（ask_user_question）
- `interaction.rs` 提供 UI 交互能力
- 用于向用户询问问题、收集偏好、澄清歧义
- 支持 1-4 个问题，每个问题 2-4 个选项
- 支持多选模式（multiSelect）
- 用户答案作为工具结果返回

### 文件预览（preview_file）
- 在专门的 UI 组件中显示文件内容或富媒体
- 支持格式：markdown、text、image、pdf、html
- 支持多文件展示：tabs（标签页）、list（列表）、grid（网格）
- 支持内联内容或 URL 引用
- 本地文件通过安全的 relay 机制访问，10 分钟过期
- 最大支持 20MB 文件

## Artifact 工作区工具

### 获取工作区（get_artifact_workspace）
- 获取当前会话的 Artifact 工作区路径和 manifest 路径
- 首次调用自动初始化目录结构
- 用于会话级 Artifact 管理

### 发布 Artifact（show_artifact）
- 将工作区中的文件显式发布为 Artifact
- 发布后更新 artifact manifest
- Sidebar 将展示该 Artifact
- 支持自定义标题、语言、预览类型等

---

## MCP 动态加载工具

### 加载 MCP 工具集（load_mcp_server）
- 根据关键词检索 MCP 工具集目录
- 返回对应工具集下的工具摘要
- 用于按需发现可用的 MCP 工具集

### 加载 MCP 工具（load_mcp_tool）
- 按关键词加载 MCP 工具到当前会话
- 返回工具的完整定义（含 description 与 parameters schema）
- 支持关键词或完整 `server::tool` 形式
- 可一次传入多个工具
- 可选限定在指定工具集下搜索

## Agent 工具（aipp:agent）

### Agent 能力
- `aipp:agent` 提供 AI Agent 内部能力支持
- 用于 Skills 加载和工作流执行
- Agent 任务处理和调度

### Skill 加载
- `load_skill` 工具加载用户定义技能的详细指令
- 支持多种来源类型：aipp（AIPP Skills）、claude_code_agents（Claude Code Agents）、claude_code_rules（Claude Code Rules）、codex（Codex）
- 支持附加文件加载

### Todo 管理集成
- `agent/todo/` 模块提供 Todo 管理能力
- AI 可直接创建、编辑、删除 Todo 任务
- Todo 状态管理与对话关联
- 任务分类和标签支持

### 动态 MCP 工具加载
- Agent 工具也集成了 `load_mcp_server` 和 `load_mcp_tool`
- 支持在 Agent 上下文中动态加载 MCP 工具
