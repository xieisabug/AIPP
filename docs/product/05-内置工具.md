# 内置工具

内置工具是 MCP 工具系统的一部分，提供了一系列开箱即用的 MCP 工具，包括 Web 搜索、文件操作、Shell 执行、Agent 能力等常用功能。内置工具与外部 MCP 服务器统一管理，均使用 `aipp:` 前缀标识。

---

## Web 搜索（多引擎）

### 搜索引擎支持
- 支持多种搜索引擎：
  - **Google** - 谷歌搜索
  - **Bing** - 必应搜索
  - **DuckDuckGo** - 隐私优先搜索引擎
  - **Kagi** - 付费搜索引擎

### 智能回退策略
- 引擎管理器 `engine_manager.rs` 管理多个搜索引擎
- 主引擎失败时自动回退到备用引擎
- 默认策略：Google → Bing 回退
- 可配置引擎优先级

### 搜索结果获取与格式化
- `handler.rs` 主搜索处理器
- 搜索结果统一格式化
- 结果摘要生成
- 链接与标题提取

## URL 内容获取

### Chrome DevTools Protocol 浏览器自动化
- `chromiumoxide/` 集成 Chrome DevTools Protocol（CDP）
- `browser_pool.rs` 浏览器池管理
- `fetcher.rs` 内容获取器
- 支持无头浏览器模式
- 页面等待与元素选择

### HTTP 客户端回退
- 浏览器获取失败时使用 HTTP 客户端回退
- reqwest 库实现 HTTP 请求
- 多种内容类型支持
- 超时与重试机制

### 内容抓取与清理
- HTML 内容解析与清理
- 去除广告和无关元素
- 主要文本内容提取
- 内容截断与格式化

### 指纹管理（反检测）
- `fingerprint.rs` 指纹管理
- 浏览器指纹随机化
- User-Agent 轮换
- 反检测措施

## Shell 执行

### Bash 命令执行
- Linux/macOS Bash 脚本执行
- `bash_ops.rs` Bash 操作实现
- 命令参数解析
- 工作目录设置

### 命令结果返回
- 标准输出捕获
- 错误输出捕获
- 退出码返回
- 执行超时控制

### 权限请求与审批
- 操作前权限确认
- `OperationPermissionDialog` 权限请求对话框
- 权限决策持久化
- 自动审批配置
- 支持三种决策：`allow`（本次允许）、`allow_and_save`（允许并记住）、`deny`（拒绝）
- ACP 工具调用权限独立确认流程（`confirm_acp_permission`）
